# Forest Fire Detection System WSN

[Forest Fire Detection](https://ffds.rishabhj.in)  

---

## Overview

**Forest Fire Detection System WSN** is an advanced web-based platform designed to monitor and detect forest fires using Wireless Sensor Networks (WSN). This system provides real-time alerts and comprehensive visualization to help prevent and manage forest fire disasters effectively.

> **Important:** This project is a proprietary website solution and **not open source software**. It is intended for deployment as a standalone web application.

---

## Key Features

- **Real-time Monitoring:** Continuously track environmental parameters through sensor data.
- **Intuitive 3D Visualization:** Interactive 3D canvas displaying forest conditions and fire alerts.
- **Detailed Logs:** Comprehensive logging panel with timestamped events and alerts.
- **User-friendly Controls:** Adjustable parameters with sleek UI controls for customization.
- **Responsive Design:** Fully optimized for desktop and mobile devices.
- **Secure & Private:** Data handled securely with no open source exposure.

---

## Technology Stack

- **Frontend:** HTML5, CSS3 (with modern CSS variables and flexbox), JavaScript
- **Fonts:** Google Fonts (Inter, JetBrains Mono)
- **Icons:** Font Awesome 6.4.0
- **Visualization:** Custom 3D canvas with radial gradients and blur effects for immersive UI
- **Design:** Dark theme with professional color palette for readability and aesthetics

---

## How to Use

1. Open the website on any modern browser (Chrome, Firefox, Edge, Safari).
2. Interact with the 3D visualization to monitor forest conditions.
3. Use the control sliders to adjust sensor sensitivity and alert thresholds.
4. Check the logs panel for detailed event history and alerts.
5. Follow on-screen mouse/touch controls for navigation and interaction.

---

## User Interface Highlights

- **Header:** Displays project title with subtle branding and a modern translucent panel.
- **Main UI:** Split layout with control panels on the left and logs on the right.
- **Panels:** Semi-transparent backgrounds with blur and shadow effects for depth.
- **Controls:** Smooth sliders with accent colors matching the theme.
- **Logs:** Monospaced font for clarity, color-coded messages for quick status recognition.
- **Responsive:** Layout adapts seamlessly to smaller screens with sticky headers and stacked panels.
